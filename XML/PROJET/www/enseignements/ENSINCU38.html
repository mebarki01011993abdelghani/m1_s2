Architecture logicielle JEE

  • Identifiant : ENSINCU38
  • Nombre de crédits 3
  • Résumé : L'objet du cours est de présenter l'architecture et le développement d'applications dans le cadre de la plateforme JEE ( Java Enterprise Edition ). Ce cours prend la suite d'un premier cours de 30 heures sur JEE. Nous étudions notamment la maîtrise des Enterprise Java Bean qui sont les composantes métier de la plateforme JEE, le couplage entre EJB et base de données relationnelle pour la manipulation des données métier ( Java Persistance API ), l'architecture MVC des applications Web avec Spring MVC et Struts, les principes de sécurisation des applications Web les technologies JSF (Java Server Faces) et GWT (Google Web Toolkit) pour la réalisation d'applications Web riches, les techniques de conception et de réalisation des Web Services (SOAP, XML-RPC, WSDL).
