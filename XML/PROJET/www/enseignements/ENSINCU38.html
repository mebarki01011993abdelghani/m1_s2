MASTER INFORMATIQUE DE MARSELLE

Architecture logicielle JEE
  • Identifiant : ENSINCU38
  • Nombre de credits 3
  • Resume : L'objet du cours est de presenter l'architecture et le developpement d'applications dans le cadre de la plateforme JEE ( Java Enterprise Edition ). Ce cours prend la suite d'un premier cours de 30 heures sur JEE. Nous etudions notamment la maitrise des Enterprise Java Bean qui sont les composantes metier de la plateforme JEE, le couplage entre EJB et base de donnees relationnelle pour la manipulation des donnees metier ( Java Persistance API ), l'architecture MVC des applications Web avec Spring MVC et Struts, les principes de securisation des applications Web les technologies JSF (Java Server Faces) et GWT (Google Web Toolkit) pour la realisation d'applications Web riches, les techniques de conception et de realisation des Web Services (SOAP, XML-RPC, WSDL).
